Guía para ejecutar esta WebApp (Sin saber programar):

  1. En tu computadora: Crea una carpeta nueva (ejemplo: "PruebaLenguaje").
  2. Crea el archivo: Abre el Bloc de Notas o cualquier editor de texto. Pega TODO este código que te he entregado.
  3. Guarda el archivo: Guárdalo dentro de la carpeta creada con el nombre index.html (asegúrate que no termine en .txt).
  4. Usando XAMPP:
    • Copia tu carpeta "PruebaLenguaje" y pégala dentro de C:/xampp/htdocs/.
    • Abre el panel de XAMPP y presiona "Start" en el módulo Apache.
    • En tu navegador, escribe: localhost/PruebaLenguaje/index.html. ¡Listo!
  5. Usando GitHub:
    • Crea un repositorio nuevo en GitHub.
    • Sube el archivo index.html.
    • Ve a Settings (Configuración) -> Pages.
    • En "Source", elige "Deploy from a branch" y selecciona "main".
    • GitHub te dará un link (ej. usuario.github.io/repositorio) para que tus alumnos entren desde cualquier lugar.
